Thawte TLS RSA CA G1 intermediate PEM approach:
- NODE_EXTRA_CA_CERTS is a Node.js env var that adds additional trust anchors
  to the TLS verification chain without disabling verification globally.
- The intermediate PEM can be fetched from DigiCert's CA repository or embedded
  as a multi-line string in the workflow step that writes it to a temp file.
- Safest pattern: write PEM to /tmp/thawte-g1.pem in a preceding step, then
  set NODE_EXTRA_CA_CERTS=/tmp/thawte-g1.pem in the run_pipeline env block.
- The Thawte TLS RSA CA G1 certificate is publicly available at:
  https://www.digicert.com/kb/digicert-root-certificates.htm
  (SHA-256: 77:EE:5A:41:B0:48:2D:6B:0C:68:54:1D:37:09:F7:66:02:F6:F7:2B:64:DE:09:4C:50:62:5B:AA:E3:12:30:FF)
