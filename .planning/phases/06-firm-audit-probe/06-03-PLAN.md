<objective>
Implement the AUDIT.md serializer per D-09 (hybrid format) and D-10 (5-value remediation enum, fail-loud enforced via TypeScript `never` exhaustiveness). The writer takes an `AuditReport` and returns a markdown string; the orchestrator (Plan 04) writes that string to disk in one atomic call.

Purpose: The AUDIT.md format is the contract Phase 7/8/9 planners read. They will `grep "enable-js-render-detail"`, `grep "fix-selector"`, `grep "migrate-to-sitemap"` against the summary table to scope their phases automatically. The format must be:
1. Stable and grep-friendly at the top (summary table)
2. Human-readable in the middle (per-firm evidence)
3. Self-describing at the bottom (run metadata for traceability)

The enum enforcement (TS `never` switches) is the build-time safety net — if a future refactor adds a 6th Remediation value without updating the writer, `pnpm typecheck` fails. This is the D-10 "writer enforces enum — no typos / new values allowed" requirement made mechanical.

    export function remediationToTargetPhase(r: Remediation): string {
      switch (r) {
        case 'enable-js-render-detail': return 'Phase 7';
        case 'fix-selector':            return 'Phase 7';
        case 'disable-firm':            return 'immediate';
        case 'migrate-to-sitemap':      return 'Phase 9';

        default: {

          throw new Error(`Unhandled remediation: ${_exhaustive as string}`);

      }
    }

    8. Main entry point:
    ```typescript
    /**
     * Serialize an AuditReport to a markdown string per Phase 6 D-09 hybrid format.
     * Caller MUST write the entire returned string in ONE writeFile call (Pitfall 6).
     */
    export function renderAuditMarkdown(report: AuditReport): string {
      const header = [
        '# Firm Audit Report',
        '',
        `_Generated by_ \`pnpm audit:firms\` _at_ ${report.runMetadata.finishedAt}`,
        '',
        'See `.planning/phases/06-firm-audit-probe/06-CONTEXT.md` for status & remediation vocabulary.',
        '',
      ].join('\n');

      const summaryTable = [
        '## Summary',
        '',
        '| Firm | Tier | Status | Items | Remediation | Target Phase |',
        '|------|------|--------|-------|-------------|--------------|',
        ...report.rows.map(summaryRowLine),
        '',
      ].join('\n');

      const evidenceBlocks = [
        '## Per-firm evidence',
        '',
        ...report.rows.map((r) => evidenceSection(r) + '\n'),
      ].join('\n');

      return [header, summaryTable, evidenceBlocks, metadataBlock(report), ''].join('\n');
    }
    ```
