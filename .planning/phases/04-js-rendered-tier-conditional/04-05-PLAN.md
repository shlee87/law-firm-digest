1. **Insert TWO new steps** immediately AFTER the existing `pnpm install --frozen-lockfile` step (currently L54) and BEFORE the `Run daily digest pipeline` step:

```yaml
      - name: Cache Playwright browser binary
        id: playwright-cache
        uses: actions/cache@v4
        with:
          path: ~/.cache/ms-playwright
          # Cache keyed on pnpm-lock hash — pnpm-lock pins the playwright
          # package version, so a playwright bump naturally invalidates the
          # cache and a fresh binary is pulled. OS prefix because Ubuntu
          # GHA runner images can change system libs in ways that require
          # re-running --with-deps.
          key: ${{ runner.os }}-playwright-${{ hashFiles('pnpm-lock.yaml') }}
          restore-keys: |
            ${{ runner.os }}-playwright-

      - name: Install Playwright chromium shell (on cache miss)
        if: steps.playwright-cache.outputs.cache-hit != 'true'
        # chromium --only-shell = ~100MB (headless shell only, no full Chromium).
        # --with-deps = install Ubuntu apt dependencies (fonts, libnss, etc)
        # without which chromium.launch() fails with "Host system is missing
        # dependencies". PROJECT.md stack lock requires this exact form.
        run: pnpm exec playwright install chromium --only-shell --with-deps
```
